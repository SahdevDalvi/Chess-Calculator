Chess Possible Moves Calculator

#Description:
This project calculates and highlights all possible legal moves for a selected chess piece.

#Features:

-Green highlight for valid moves.
-Red highlight for enemy captures.
-Utilizes provided helper functions for smooth board interaction.

#Main Functionality:
***When a player selects a chess piece:

-The script clears previous highlights.
-It checks all valid moves based on the piece type.
-Highlights empty tiles green and enemy tiles red.

#Coding Practices:

-Written using OOP, design patterns, and good coding standards.
-Structured for easy extension and maintenance.

#Tools Used:

-Unity 2020.3.40f1
-C#
